# scrapers
List of scrapers for many Irish Job site boards.

Including:
- Indeed.ie
- Monster.ie
- IrishJobs.ie
- Computerjobs.ie
- and more to come...

All these scripts work with Python3. I'm using the BeautifulSoup library as well as pandas.

Beautifulsoup installation:

pip3 install bs4

Pandas installation:

pip3 install pandas
